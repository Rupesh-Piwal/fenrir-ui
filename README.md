# Fenrir UI - aps Dashboard

Dashboard web-interface for Fenrir Security

## Tech Stack

- **Framework**: React 18+ with TypeScript
- **Bundler**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **State Management**: React Context & Hooks

## Getting Started

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Rupesh-Piwal/fenrir-ui.git
   cd fenrir-ui
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `src/components`: Reusable UI components.
- `src/pages`: Top-level page components and layouts.
- `src/data`: Mock data structures for development and demonstration.
- `src/context`: Global application state (Theme).

## Known Limitations

- **Mock Data:**  
  I have used static JSON (fake) data to show scan results and logs.  
   Because I don’t have full product awareness yet, I used mock data to focus on building and rendering the UI properly.
